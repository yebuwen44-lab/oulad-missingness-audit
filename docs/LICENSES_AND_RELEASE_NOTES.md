## Release status

This repository contains the processed analytical table, frozen numerical results, source code, configurations, figures, and integrity-verification utilities associated with the manuscript.

The frozen results record the values reported in the manuscript. Newly generated rerun outputs are written separately and do not overwrite the frozen record.

The repository contains no API keys, passwords, private correspondence, names, contact details, or direct personal identifiers. The OULAD learner identifier is an anonymised dataset key inherited from the public source.
