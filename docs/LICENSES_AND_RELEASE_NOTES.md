# Licenses and release notes

## OULAD-derived data

The processed analytical table is derived from the Open University Learning Analytics Dataset (OULAD). OULAD is released under the Creative Commons Attribution 4.0 International license (CC BY 4.0). Redistribution and adaptation require appropriate attribution to the original dataset creators and citation of the dataset paper.

## Analysis code

No public software license has yet been selected by the manuscript authors. Before publishing this package on GitHub or Zenodo, select an explicit code license. The MIT License is a practical default for reproducibility code, but the final choice must be made by the authors.

## Release status

This package is a repository-ready candidate assembled from the frozen v34 manuscript assets. It contains no API keys, passwords, private correspondence, or direct personal identifiers. The OULAD student identifier is an anonymised dataset key inherited from the public source.
